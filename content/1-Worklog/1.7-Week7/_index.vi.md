### Kết quả đạt được tuần 7:

* Hiểu cách triển khai Amazon S3.

* Thực hành thành công các tính năng của Amazon S3:
  * Tạo S3 Bucket và tải dữ liệu lên hệ thống.
  * Cấu hình Static Website Hosting.
  * Cấu hình Public Access.
  * Tăng tốc truy cập với CloudFront.
  * Quản lý Versioning.
  * Di chuyển và sao chép đối tượng giữa các Region.

* Hiểu vai trò và cơ chế hoạt động của AWS Storage Gateway.

* Thực hành thành công triển khai AWS Storage Gateway:
  * Tạo Storage Gateway.
  * Tạo File Share.
  * Gắn File Share trên máy On-premises.
